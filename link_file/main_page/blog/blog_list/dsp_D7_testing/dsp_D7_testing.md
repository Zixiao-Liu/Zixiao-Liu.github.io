1. 抓log工具  
    分为三个音频文件：麦克风的总收音、reference文件、处理过后输出的音频。  
    * 麦克风总收音文件：麦克风把能录的声音都录入，包括人声源，喇叭声源，上下盖、周围盖震动等自身震动产生的声音等等（通过空气、电路板等）。  
    * reference文件：喇叭声源（自身喇叭发声，自身麦克风收音的情况）
    * 处理后输出的音频文件： 麦克风总收音减去eference音频所得的文件（只剩下人声和杂音）杂音会导致失真
